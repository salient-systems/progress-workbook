The Progress Workbook, developed by the Computer Science senior project team Salient Systems, is designed to allow teachers and administrators to track and analyze the performance of students, cohorts, and classes using a highly customizable graphing system.

Assessments and performance graphs comprise the key aspects of the system, and everything else is designed to support and complement this functionality. Each class can have multiple assessments, which in turn have multiple instances of an assessment. For example, a particular class might have a Learning Check assessment which is conducted once a week throughout the term, as well as four Lab Reports and two Quizzes. This means that the class would have three assessments which are conducted at different times and various cadences. The results of each assessment are entered on the Assessment page, much like an Excel spreadsheet. Users can then select a class, student, user, cohort, or any grouping thereof and plot their performance on the Performance page.


You can access the Progress Workbook by entering the appropriate domain name or IP address in your web browser's address bar. The URL might vary depending on your setup, so please ask an administrator if you don't know what it is.

Tip

We recommend adding the Progress Workbook to your favorites for easy access.

The first thing you'll see when you access the website is a login screen asking for your username and password. All new user accounts must be created by an administrator, so you'll need to make sure your account has been created before attempting to log in. If you've lost your password or can't remember it, please ask your administrator to reset the password on your account.

Once you've entered your account information, click the "Sign in" button to access the system.


Class List

The Classes page displays a list of all of the classes in the Progress Workbook. The view defaults to the current term, but classes from prior years/semesters/quarters/etc. can be accessed via the term drop-down menu at the bottom of the page. The class list is sorted by name, but you can optionally sort by subject, period, grade, or teacher by clicking on the column headers.

Note

Administrators can see all of the classes in the system, while teachers only have access to their own classes.

Searching

While the class list could become very long, you can quickly find a specific class by typing its name in the search bar at the top of the page. This filters the results to show only the classes, subjects, periods, grades, or teachers which match your search.

Performing Actions on Multiple Classes

At any time, you can select the checkboxes next to one or more classes, allowing you to compare them on the Performance page or delete them from the system.

Creating a New Class

To add a new class to the system, click on the "Create Class" button in the upper-right corner of the Classes page. This brings up a special window where you'll enter the name of the class, its subject, the target grade level, and, optionally, the period.

Note

Classes are automatically assigned to the current user. Additionally, administrators can create new subjects on the Settings page.

Once you're happy with your selection, click "Create" to add the class to the system.

Class Details

Selecting a class from the Classes page will display the enrolled students, as well as any assessments that have been created. The assessments are displayed in the order they were created, while the student list can be sorted by first or last name, student ID, or grade level.

Editing the Class

To edit the class details, simply click the "Edit" button in the top-right corner of the Class Details page. This will display an edit window that is virtually identical to the one for creating a class. From here, you can rename the class or change its subject, period, or grade level. When you're done, simply click "Update" and your changes will be instantly reflected on the page.

Adding a Student to the Class

To add a student to the class, click the "Add Student" button in the upper-right corner of the Class Details page. This will display a search box, letting you find and add any active student in the system. Simply begin typing the student's name or student ID to see a list of matching results. Selecting a student will instantly add them to the class.

Performing Actions on Multiple Students

At any time, you can select the checkboxes next to one or more students, allowing you to compare them on the Performance page, add them to a cohort, or remove them from the class via the Actions menu. If you choose to add the student(s) to a cohort, a window will appear with a list of all of the cohorts in the system. Simply select a cohort and click "Add" to complete the process.

Creating a New Assessment

To create a new assessment, click on the "Create Assessment" button above the list of assessments. This brings up a window allowing you to choose the name and style for the assessment. Assessments come in three styles: Run Chart, Criteria-Based, and Standards-Based. Which one you choose depends on how you want to view and manage the grades. Click "Create" to add the assessment to the class.

Assessment Styles

Run Chart
A Run Chart contains a list assessments laid out horizontally. Each assessment has a single criterion, making it ideal for assessments which are given at a regular cadence and have only one result (for example, a Learning Check).
Criteria-Based
A Criteria-Based assessment type is best suited for assessments which use the same set of criteria over and over, such as a Lab Report. Assessments are aligned horizontally across the top, and the criteria are displayed next to the names of each student.
Standards-Based
Standards-Based assessment types work well in situations where there are multiple assessments with a varying number of criteria (such as an "I Can..."). The students are arranged horizontally across the top of the page, while the assessments and criteria are grouped vertically.

To enter grades for an assessment, select the appropriate assessment type on the Class Details page. This will display an Excel-like spreadsheet containing the list of students, assessments, and any applicable criteria in one of three styles: Run Chart, Criteria-Based, and Standards-Based.

The grades for each assessment or criterion are color-coded, allowing you to quickly see at-a-glance which students are doing well or are underperforming. The darker the shade of blue, the better the grade. To add or update a grade, simply select the cell you want to edit and enter a value. The grade will be automatically saved and the cell color will update accordingly.

The three assessment layouts also contain summary grades, allowing you to see how a student is performing overall or how a class is performing on a single assessment or criterion. Summary grades greater than one and less than 75% are displayed in red, indicating that the student needs assistance or the entire class is struggling. Average grades between 75% and 85% are shown in yellow, and grades greater than 85% are indicated in green.

Tip

Hover over a student’s name to see a graph of their progress compared to the rest of the class.

Editing an Assessment

By default, new assessments contain a single assessment with a single criterion. To add additional assessments and criteria, simply click the "Edit" button in the top-right corner of the page. This will open a special window which allows you to add, update, or delete the assessments and criteria. In most cases, each assessment may have multiple criteria, and each criterion has a maximum value (which is used to calculate the percentages). The exact options may vary based on the assessment style. When you’re finished, click "Save"" to apply your changes and refresh the page.

Deleting an Assessment

To delete an assessment, click on the drop-down arrow next to the assessment name on the Class Details page and select the "Delete" option. After confirming your decision, the assessment type will be removed.

Assessment Quick Graph

To quickly graph an assessment, click on the drop-down arrow next to the assessment name on the Class Details page and select the "Show on Graph" option. The assessment’s graph will be displayed to the right of the Assessments list.

To edit the student's profile, simply click the "Edit" button in the top-right corner of the Student Profile page. This will display an edit window that is virtually identical to the one for creating a student. From here, you can change the student's first and last name, student ID, grade level, and gender. You can also activate or deactivate the student using the checkboxes. When you're done, simply click "Update" and your changes will be instantly reflected on the page.

The Performance page is a "data playground" of sorts. The central focus is the graph, which runs across the top of the page. While the graph is initially empty, you can plot the system's data in a virtually endless number of configurations using the data sets.

Using Data Sets

Each piece of data on the graph has an associated data set which has been color-coded appropriately. Data sets are highly customizable, allowing you to compare any number of combinations of students, classes, teachers, cohorts, assessments, criteria, and more.

In order to plot a data set on the graph, you'll need to select a term, class, and statistic (see section 9.1.1 for more details). At this point, additional options for the data set will begin to unlock, including the assessment type, assessment, and criterion. To narrow down the options for a specific student, cohort, or teacher, simply search for a name and select the desired person or group. Once you're happy with your selection, click "Save" to plot the data on the graph. Additional data sets can be added using the "Add Data Set" button in the lower-right corner of the page. To minimize a data set and reduce the amount of screen real estate it takes up, simply click on its name.

Note

The available options in the data set will vary based on your current selection, and only administrators can search for a teacher.

Statistics

Statistics allow you to choose how the data appears on the graph. You can combine data sets with various statistics to look at performance data from multiple angles.

Total Correct
A raw score which is the sum of the grades.
Percentage Correct
The percentage of correct answers.
Total Possible
The maximum points possible based on the number of students present.
Total Goal
The maximum points possible if every single student is present.
Percent of Term
A linear representation of the expected progress throughout the term. Starts at 0 for the first assessment and ends at the maximum possible value by the last assessment. Often used to determine where a student is expected to be at any given point during a term (assuming a linear progression).
Score Dist. (Total)
The number of students who received each grade for a given assessment (e.g. three students received a 1, two students received a 2, five students received a 3, etc.).
Score Dist. (Percent)
The percentage of students who received each grade for a given assessment (e.g. 30% of students received a 1, 20% of students received a 2, 50% of students received a 3, etc.).
Students Present
The number of students present for a given assessment.
Students Enrolled
The number of students enrolled in a given class.

Common Configurations

There are a virtually infinite number of possible data set configurations. Here are a few ideas to get you started:

Compare the Actual Grade of a Class to Its Potential Grade
Data Set 1: Select a class and choose the "Total Correct" statistic.
Data Set 2: Select the same class and choose the "Total Possible" statistic.
View the Overall Grade for an Assessment Type
Data Set 1: Select a class and assessment type, then and choose the "Percentage Correct" statistic.
View a Particular Score for a Student in an Assessment
Data Set 1: Search for the student and select a class, assessment type, and assessment. Then choose the "Total Correct" statistic.
Compare a Cohort to a Class
Data Set 1: Select a class and choose the "Percentage Correct" statistic.
Data Set 2: Search for a cohort and select a class. Then choose the "Percentage Correct" statistic.
Compare the Performance of Two Students on a Particular Criterion
Data Set 1: Search for the first student, then select a class, assessment type, assessment, and criterion. Then choose the "Total Correct" statistic.
Data Set 2: Search for the second student, then select a class, assessment type, assessment, and criterion. Then choose the "Total Correct" statistic.

Automatically Configuring the Performance Graph

Selecting the "Compare" action on any page will automatically navigate to the Performance page and configure the data sets appropriately. This makes it easy to quickly compare students, classes, cohorts, and more. You can, of course, tweak the pre-configured data sets as needed.


The Settings page allows users to change their password. Administrators can also access additional options like importing students from a CSV file, adding subjects, and advancing the term.

Changing Your Account Password

To change your password, enter your current password followed by your desired new password. Confirm your choice to prevent a typographical error, then select the "Change" button.

Advancing the Term

At the end of a term (e.g. year, semester, quarter, etc.), administrators have the option of archiving the current classes and creating a new term. Archived classes can always be accessed via the term drop-down menu on any page which displays a list of classes.

When advancing the term, admins can also choose to advance the grade level of every active student in the system. This is designed to be used once a year, saving teachers and administrators the time and energy required to manually update the grade for each enrolled student. If a student's grade is at the maximum grade level, their profile is deactivated and their grade is left untouched. Inactive students can be found by clicking on the "Inactive Students" button on the Students page.

Creating New Subjects

When a user creates a class (section 4.1.3), they are required to choose a subject. To add a new subject, an administrator must click the "Add Subject" button on the Settings page, enter the subject name, and select "Create."

Importing Students

To save time during the initial setup or when a large group of new students enter the school, administrators can import a comma separated values (CSV) file with the student details. The CSV must include a header row containing column names. Columns may be in any order, but the columns must be named as given in parentheses. The following columns are mandatory: first name (fname), last name (lname), grade level (grade_level), true or false depending on if the student is active (is_active) and student ID (sid). The following column is optional: gender (gender). Mandatory columns must contain a non-blank value for each student. If values are missing, the import process will fail and display an error. All students prior to the failing record will be imported, but the failing record and all subsequent records in the file will not.

Note

It is recommend that you ensure that each student ID is unique. Additionally, importing students that already exist in the system will create duplicates.


How do I create a new account?

User accounts can be created on the Users page. To create a new account, simply click the "Create User" button in the upper-right corner, fill out the form, and submit it. The user will now have access to the system.

I can't remember my password or access my account

If you can't access your account, please check with your administrator to ensure that the account is active. They'll also be able to reset your password if necessary.

How do I edit a class/student/user/cohort?

Navigate to the details page (in the case of classes and cohorts) or the profile page (in the case of students and users) and click the "Edit" button in the top-right corner.

How do I delete a class/student/user/cohort/assessment?

Navigate to the Class, Student, User, Cohort, or Class Details page and select the checkboxes next to the items you want to delete. In the "Actions" drop-down menu in the lower-left corner, click "Delete" and then confirm your choice.

Why aren't any classes listed on a student/user's profile?

Most likely, the student or user doesn't have any classes associated with their profile. To fix this, add the student to a class or have the teacher create a new class. If you're sure the profile has an associated class, try changing the term using the drop-down menu at the bottom of the page.

Why can't I find a specific student/user?

If you can't find a student or user, chances are their profile was deactivated. Click the "Show Inactive" button at the bottom of the page to view the list of inactive students or users. There's also a possibility, however unlikely, that they were deleted from the system.

What's the difference between a teacher and an admin

Teachers are able to create and manage cohorts and their own classes, add new students to the system, and manage their account. Administrators can do all this and more, including managing users, deleting students, viewing the classes of other teachers, and managing various system settings.

What's the difference between an assessment and an assessment type?

An assessment type consists of multiple assessments. Most classes have a number of assessment types, such as exams, quizzes, papers, lab reports, etc. Each assessment type has a different purpose, and they are distributed at different times and intervals throughout the term. An assessment is a single instance (or distribution) of an assessment type, so a midterm and a final are two different assessments in the exams assessment type.
